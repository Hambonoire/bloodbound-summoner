# Bloodbound Summoner

A JavaScript roguelike card game prototype built around risky self-damage, sacrifice, and custom summoning.

## Overview

In this game, each run starts with a limited deck.
Every time you win a match, you gain access to a randomized pack from a larger pool.
The goal is to build stronger summon combinations through risky choices and long-term deck shaping.

## Core ideas

- Limited starting deck
- Run-based progression
- Randomized pack rewards after victories
- Summoning as the main combat identity
- Masochistic mechanics like self-damage, sacrifice, and curse tradeoffs
